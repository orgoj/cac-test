# Documentation

This directory contains documentation for various tools and issues encountered in the Claude Code environment.

## Contents

### [PRPM_PROXY_ISSUE.md](PRPM_PROXY_ISSUE.md)
Detailed analysis of the DNS resolution issue when using PRPM (Package Manager for AI Prompts) in the Claude Code container environment. This document:
- Describes the root cause (DNS lookup before proxy application)
- Provides technical analysis and verification tests
- Proposes solutions for PRPM maintainers
- Includes workarounds and issue template for reporting

This documentation can be used as a basis for creating an issue in the PRPM repository.
